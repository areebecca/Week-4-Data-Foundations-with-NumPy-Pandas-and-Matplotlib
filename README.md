## Lab Title: Data Foundations with NumPy, Pandas, and Matplotlib
Integrated Data Manipulation and Visualization in Python 
## Objective
This lab aims to provide hands-on experience in the core pillars of Python data science:

NumPy: Efficiently creating and manipulating numerical arrays.

Pandas: Executing complex data merging operations between multiple datasets.

Matplotlib: Visualizing data distributions through statistical plotting.
## Tools Used
Python 3.x: Primary programming environment.

NumPy: For fundamental numerical operations and array handling.

Pandas: For structured data manipulation and relational-style merging.

Matplotlib: For generating graphical representations of data.
## Step-by-Step Process
Array Engineering: Initialized 1D and 2D NumPy arrays to handle vectorized numerical operations.

Relational Merging: Constructed multiple DataFrames and executed Inner, Outer, Left, and Right joins to synchronize records across different sources.

Distribution Analysis: Used Histograms and Box Plots to identify data spread, skewness, and outliers.

Categorical & Part-to-Whole Comparison: Implemented Bar Graphs for discrete comparisons and Pie Charts for proportional analysis.

Trend & Relationship Mapping: Developed Line Graphs for time-series/sequential trends and Scatter Plots to identify correlations between variables.

Advanced Density Estimation: Created Violin Plots and Heat Maps to visualize complex distributions and matrix correlations.

## Key Observations:
Merge Integrity: Learned that selecting the correct join type (e.g., Left vs. Inner) is vital to prevent unintentional data loss in fintech and reporting contexts.

Visualization Selection:

Box Plots are superior for spotting outliers, while Violin Plots provide a deeper look at the probability density.

Heat Maps effectively simplify large correlation matrices into intuitive color-coded insights.

NumPy vs. Lists: Confirmed that NumPy's vectorized operations significantly reduce code complexity and execution time compared to standard Python loops.

Data Storytelling: Discovered that a Line Graph is most effective for showing progress over time, whereas Scatter Plots are essential for identifying variable dependencies.
